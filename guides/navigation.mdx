## Overview

The Monad website navigation bar contains four primary sections: Home, Explore, Build, and Resources. Each section (except Home) has a dropdown menu with categorized links. This guide documents the full navigation structure.

## Navigation Layout

The top navigation bar links to:

- **Home** — [monad.xyz](https://monad.xyz)
- **Explore** — Dropdown with media, announcements, events, and block explorers
- **Build** — Dropdown with developer resources, infrastructure, and programs
- **Institutions** — [monad.xyz/institutions](https://monad.xyz/institutions)
- **Resources** — Dropdown with foundation links, jobs, branding, and socials

## Explore Dropdown

The Explore dropdown provides access to Monad content and block explorers.

| Category | Links |
|---|---|
| Content | Monad Media, Announcements, Events |
| Block Explorers | MonadVision, Monadscan, Gmonads |

**External links:**
- MonadVision — [monadvision.com](https://monadvision.com)
