# Keyboard Shortcuts

Monad.xyz is primarily a marketing and informational site with limited interactive elements. The keyboard interactions below reflect what is available.

## Site-Specific Interactions

| Shortcut | Action | Context |
|---|---|---|
| Sidebar dots (click) | Scroll to section | Landing page section navigation |
| `Escape` | Close dropdown menus | Navigation dropdowns and modals |

## Standard Browser Navigation

These standard keyboard shortcuts work across monad.xyz as they do on any web page.

| Shortcut | Action |
|---|---|
| `Tab` | Move focus to the next interactive element |
| `Shift + Tab` | Move focus to the previous interactive element |
| `Enter` | Activate the focused link or button |
| `Space` | Scroll down the page, or activate a focused button |
| `Home` | Scroll to the top of the page |
| `End` | Scroll to the bottom of the page |
| `Ctrl/Cmd + F` | Open browser find-in-page |
| `Ctrl/Cmd + L` | Focus the browser address bar |
| `Alt/Cmd + Left Arrow` | Navigate back |
| `Alt/Cmd + Right Arrow` | Navigate forward |

## Accessibility Notes

- All navigation links and buttons are reachable via `Tab`.
- Dropdown menus can be dismissed with `Escape`.
- The site does not implement custom keyboard shortcuts beyond standard browser behavior.
