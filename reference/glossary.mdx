# Glossary

An alphabetical reference of terms used throughout the Monad ecosystem and this documentation.

---

### Block Time

The interval between consecutive blocks on the Monad network. Monad targets 0.4-second block times. See [Home Page Guide](/guides/home).

### DeltaV

An accelerator program operated by Monad to support early-stage projects building on the network. Accessible at `deltav.monad.xyz`. See [Ecosystem Directory](/guides/ecosystem).

### EVM (Ethereum Virtual Machine)

The execution environment for smart contracts. Monad is 100% EVM-compatible, meaning existing Ethereum contracts and tooling work without modification. See [Build on Monad](/guides/build).

### Finality

The point at which a transaction is irreversible. Monad achieves 0.8-second finality. See [Home Page Guide](/guides/home).

### Gmonads

A community initiative within the Monad ecosystem. Accessible at `gmonads.com`. See [Ecosystem Directory](/guides/ecosystem).

### Mach Accelerator

A program for teams building infrastructure and applications on Monad. Accessible at `mach.monad.xyz`. See [Ecosystem Directory](/guides/ecosystem).

### Monad

A Layer 1 blockchain that delivers 10,000 TPS with full EVM compatibility. It uses parallel execution, MonadBFT consensus, and MonadDB for storage. See [Getting Started](/docs/getting-started).

### MonadBFT

The Byzantine Fault Tolerant consensus mechanism used by Monad. It enables agreement among 200+ validators with sub-second finality. See [Home Page Guide](/guides/home).

### MonadDB

The custom database layer designed for Monad's parallel execution engine. It provides the storage backend that enables high throughput. See [Home Page Guide](/guides/home).

### Monad Foundation

The organization responsible for stewarding the development and growth of the Monad protocol and ecosystem. See [Announcements](/guides/announcements).

### Monad Madness

A competition event for builders in the Monad ecosystem. Accessible at `madness.monad.xyz`. See [Events](/guides/events).

### Monad Memo

The official newsletter published by the Monad team, covering protocol updates, ecosystem news, and community highlights. See [Newsletter](/guides/newsletter).

### Monad Momentum

A program designed to support and grow the Monad community. Accessible at `momentum.monad.xyz`. See [Ecosystem Directory](/guides/ecosystem).

### Monadscan

The block explorer for the Monad network. Accessible at `monadscan.com`. See [Infrastructure Directory](/guides/infra-directory).

### MonadVision

A conference and event series hosted by the Monad ecosystem. Accessible at `monadvision.com`. See [Events](/guides/events).

### Parallel Execution

A technique where multiple transactions are processed simultaneously rather than sequentially. This is a core component of how Monad achieves 10,000 TPS. See [Home Page Guide](/guides/home).

### RaptorCast

The data dissemination protocol used in Monad's consensus layer to propagate blocks efficiently across 200+ validators. See [Home Page Guide](/guides/home).

### TPS (Transactions Per Second)

A measure of network throughput. Monad targets 10,000 TPS. See [Home Page Guide](/guides/home).

### Validator

A node operator that participates in MonadBFT consensus. Monad supports 200+ validators. See [Permissions](/reference/permissions).
