## Overview

Monad is a Layer 1 blockchain designed for high throughput while maintaining full compatibility with the Ethereum Virtual Machine (EVM). This page covers the core concepts, performance characteristics, and recommended starting points.

## Core Concepts

### EVM Compatibility

Monad is 100% EVM-compatible. Existing Ethereum smart contracts, tools, and wallets work on Monad without modification. Developers can deploy Solidity contracts and use standard Ethereum tooling (Hardhat, Foundry, ethers.js) directly.

### Throughput (TPS)

Monad processes 10,000 transactions per second. This is achieved through pipelined execution and optimistic parallel processing of transactions within each block.

### Finality

Transactions on Monad reach finality in 0.8 seconds. Block times are 0.4 seconds, meaning two blocks are produced per second.

### Validators

The network operates with 200+ validators, supporting decentralization without sacrificing performance. Monad uses a BFT-based consensus mechanism with RaptorCast for efficient block propagation.
